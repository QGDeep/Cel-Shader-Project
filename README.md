# Cel-Shader-Project
A Cel Shader Project that implements the Cel Shader effect using various methods.

## Features
- Simple Cel Shader - uses heavy post process effects 
- Advanced Cel Shader - uses heavy post process effects (Planned)
- Advanced Cel Shader - uses various materials (Planned)
- Toon Shader - uses various materials (Planned)

## Technologies
- Unreal Engine 5
- Blueprints
- Material Blueprint
